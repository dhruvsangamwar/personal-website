# Dhruv's Portfolio

This is a porfolio complete with a blog. Includes:

- MDX and Markdown support
- Optimized for SEO (sitemap, robots, JSON-LD schema)
- RSS Feed
- Dynamic OG images
- Syntax highlighting
- Tailwind v4
- Vercel Speed Insights / Web Analytics
- Geist font

### Clone and Deploy

Clone this repository and just run: 

```bash
pnpm dev
```
